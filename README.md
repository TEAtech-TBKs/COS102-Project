# COS102 Problem Solving Project

## Overview

This repository contains solutions to assigned problems in the COS102 (Problem Solving) course. The project focuses on applying algorithmic thinking using pseudocode, flowcharts, and Python programming.

---

## Project Tasks

### 1. Right-Angled Triangle Checker

A program that takes two angles of a triangle as input, calculates the third angle, and determines whether the triangle is right-angled (i.e., one angle equals 90°).

### 2. Palindrome Checker

A program that checks whether a given number is a palindrome by reversing the number and comparing it with the original value.

---

## Tools & Concepts Used

* Python programming
* Pseudocode design
* Flowchart representation
* Conditional statements (`if`, `else`)
* Loops (`while`)
* Mathematical operations (modulus and integer division)

---

## How to Run the Programs

1. Make sure Python is installed on your system.
2. Open your terminal or command prompt.
3. Navigate to the project folder.
4. Run the desired file:

```
python triangle.py
```

or

```
python palindrome.py
```
